## Arquivos

| Questão   | Descrição                                                            | Arquivo                        |
| --------- | -------------------------------------------------------------------- | ------------------------------ |
| Questão 1 | Solução desenvolvida em Jupyter Notebook                             | [Acessar Q1.ipynb](./Q1.ipynb) |

## Execução

Para executar os notebooks, abra os arquivos `.ipynb` utilizando o Jupyter Notebook, JupyterLab ou Visual Studio Code.

Para executar o arquivo da Questão 2, utilize:

```bash
python Q2.py
```

Antes da execução, verifique se todas as dependências necessárias estão instaladas no ambiente Python.
